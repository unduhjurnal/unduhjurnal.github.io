Jawaban:

Langkah-langkah yang dilakukan oleh Rudi adalah :

Mengecek Properties file tersebut, apakah ditujukan untuk sistem operasinya atau tidak.Mengecek website tempat ia mendapatkan file tersebut, dan mencari informasi kepada sistem operasi apa file tersebut ditujukan.Mengunduh ulang file yang terdapat di website tersebut, dan mencocokan dengan sistem operasi yang dimiliki oleh Rudi.Penjelasan:

Kesalahan sistem operasi saat mendeteksi aplikasi untuk membuka file, biasanya terjadi karena sistem operasi tersebut tidak men-support jenis ekstensi dari file yang ada. Karena, setiap sistem operasi sudah memiliki ekstensinya tersendiri.

Apa itu ekstensi?

Ekstensi adalah format suatu file, biasanya terdapat di akhir nama file, dan didahului dengan tanda titik (.)

Contohnya : Brainly.exe

(.exe adalah ekstensi dari Windows, dan digunakan untuk menjalankan suatu aplikasi/program).
