Vol. kerucut = 5.544 cm³ dan tinggi (t) = 27 cm

#menentukan panjang jari jari (r) kerucut

Vol. kerucut = 1/3 x πr² x t

5.544cm³ = 1/3 x 22/7 x r² x 27cm

5.544cm³ = 22/7 x r² x 9cm

5.544cm³ x 7 = 22 x r² x 9cm

38.808cm³ = 198cm x r²

r² = 38.808cm³/198cm

r² = 196cm²

r = √196cm² = √(14cm)² = 14cm

Jadi, panjang jari-jari (r) kerucut tersebut adalah 14cm.
