7. Tuliskan dan gambarkan dalam tabel sistem Flowchart dan program Flowchart beserta Keterangannya.​


Jawaban :

Fungsi flowchart adalah digunakan untuk menganalisa, mendesain, mendokumentasikan dan memanajemen sebuah proses atau program di berbagai bidang. Secara khusus, flowchart berfungsi untuk membantu menggambarkan situasi apa yang sedang terjadi dan yang akan terjadi dari sebuah simbol dan tanda penghubungnya. Selain itu, flowchart ini mampu memperjelas sebuah alur dari suatu sistem baik itu kekurangan atau kelebihan dari berbagai proses di dalam tahapan suatu sistem.

Penjelasan:

Tujuan Flowchart

Berikut ini terdapat beberapa tujuan flowchart, diantaranya adalah:

Menggambarkan urutan atau tahapan dari penyelesaian masalah

Menggambarkan permasalahan secara sederhana, terurai, rapi dan jelas

Flowchart membantu analis dan programmer untuk memecahkan masalah ke dalam bagian-bagian yang lebih kecil dan menolong dalam menganalisis alternatif-alternatif lain dalam pengoperasian. Flowchart mempermudah penyelesaian suatu masalah khususnya masalah yang perlu dipelajari dan dievaluasi lebih lanjut.

Siklus dalam flowchart adalah siklus input-proses-output atau sering disebut siklus IPO. Siklus IPO dapat kita analogikan sebagai komponen hardware komputer. Komponen input seperti keyboard, mouse, scanner, dan sebagainya dapat kita andaikan sebagai input (penerimaan suatu perintah atau data yang akan diproses).

Komponen proses seperti CPU akan mengolah/memproses data yang telah masuk dalam komponen input agar menghasilkan suatu hasil. Dan akhirnya hasil pemrosesan data atau perintah yang dilakukan oleh CPU ditampilkan pada monitor, speaker atau printer.

Simbol-simbol yang di pakai dalam flowchart dibagi menjadi 3 kelompok :

1. Flow Direction Symbols

2. Processing Symbols

3. Simbol Input atau Output

Peristiwa dengan mengunakan Flowchart (Diagram Alir) dan Pseudocode contohnya adalah menghitung luas segitiga, keliling persegi,luas lingkaran

Pembahasan

Flowchart merupakan diagram yang menggambarkan alur kejadian suatu peristiwa, sedangkan pseudocode merupakan code program sementara yang mengikuti alur kejadian yang belum teruji kebenarannya.

pseudocode menghitung luas segitiga

start

Deklarasi

luas, alas, tinggi as integer

Proses

  Read(alas, tinggi)

  luas = 0.5 * alas * tinggi

  write (luas)

end

Pseudocode menghitung keliling persegi

start

Deklarasi

keliling, sisi as integer

Proses

  Read(sisi)

  keliling = 4 * sisi

  write (keliling)

end

Pseudocode menghitung luas lignkaran

start

Deklarasi

luas, r as integer

Proses

  Read(r)

  luas = 3.14 * r * r

  write (luas)

end

14. Buatlah 3 contoh peristiwa dengan mengunakan Flowchart (Diagram Alir)dan Pseudocode


1. Memasak Telur

Kamu ingin memasak telur, ada variasi langkah flowchart yang bisa dilakukan:

Siapkan telur yang akan dimasak

Tentukan telurnya akan dimasak seperti apa

Kalau ingin dimasak orak-arik, maka:

3.1. Pecahkan telurnya ke dalam wadah

3.2. Kocok telur dan tambahkan bumbu sesuai selera

3.3. Panaskan penggorengan

3.4. Masukkan kocokan telur ke penggorengan yang sudah panas

3.5. Aduk telur sampai menjadi orak-arik

4. Kalau ingin dimasak ceplok, maka:

4.1. Panaskan penggorengan yang sudah diisi minyak

4.2. Pecahkan telur di penggorengan dan tunggu sampai matang

Kalau ingin dimasak rebus, maka: 5.1. Didihkan air dalam panci

5.2. Masukkan telur utuh ke dalamnya dan tunggu beberapa menit sampai isinya matang

Siap untuk dimakan

2. Menyalakan Komputer

Kamu akan menyalakan komputer, alur flowchart-nya:

Pencet tombol “on” di CPU

Apakah komputer sudah menyala normal?

Jika ya 3.1. Apakah muncul pesan error di layar?

3.1.1. Jika ya, lakukan troubleshooting

3.1.2. Jika tidak, komputer bisa langsung digunakan

Jika tidak4.1. Apakah lampu indikator menyala

4.1.1. Jika ya, pencet tombol “on” pada monitor

4.1.2. Jika tidak, cek sumber listrik untuk memastikan arus

4.2. Apakah masalah teratasi?

4.2.1. Jika ya, komputer bisa langsung digunakan

4.2.2. Jika tidak, silakan panggil teknisi untuk memperbaikinya

3. Memilah Sampah

Untuk merawat bumi, pemilahan sampah wajib dilakukan sebelum dibawa ke pembuangan akhir. Alur flowchart-nya kurang lebih:

Pilah sampah berdasarkan jenisnyaJenis pertama adalah sampah organikSampah organik dapat diolah menjadi pupukJenis kedua adalah sampah yang dapat digunakan kembaliSampah jenis ini dapat dimanfaatkan untuk hal lainSampah ketiga adalah sampah yang dapat didaur ulangSampah ini dapat didaur ulang menjadi barang lainJika tidak tergolong ke sampah organik, sampah yang dapat digunakan kembali, dan sampah yang dapat didaur ulang, baru dibuang ke tempat pembuangan akhir.

23. Perbedaan flowchart sistem dan flowchart program


• Flowchart Sistem merupakan bagan yang menunjukkan alur kerja atau apa yang sedang dikerjakan di dalam sistem secara keseluruhan dan menjelaskan urutan dari prosedur-prosedur yang ada di dalam sistem. Dengan kata lain, flowchart ini merupakan dekripsi secara grafik dari urutan prosedur-prosedur yang terkombinasi yang membentuk suatu sistem.

• Bagan alir program (program flowchart) merupakan bagan yang menjelaskan secara rinci langkah-langkah dari proses program. Bagan alir program dibuat dari derivikasi bagan alir sistem.

24. Tuliskan perbedaan flowchart sistem dan flowchart dokumen


Perbedaan antara flowchart sistem dengan flowchart dokumen sebagai berikut:

Flowchart Sistem : diagram yang menunjukkan alur kerja dari keseluruhan  sistem. Menunjukkan serangkaian operasi pada sistem. Diagram alir sistem menunjukkan apa yang terjadi dalam sistem. Diagram alir sistem terdiri dari kebocoran data dari sistem dan proses yang mencakup perubahan pada data tersebut.

Flowchart Dokumen : Bagan alir dokumen atau yang biasa disebut dengan bagan alir formulir adalah bagan alir yang menunjukkan alur laporan dan formulir, termasuk salinannya. Bagan alir dokumen ini memiliki simbol yang sama dengan yang digunakan dalam bagan alir sistem.

Pembahasan:

Pada dasarnya, diagram alir adalah diagram yang berisi urutan proses dan simbol-simbol tertentu yang merinci hubungan antara satu proses dengan proses lainnya  dalam  sistem.

Fungsi flowchart adalah untuk merepresentasikan Algoritma, memodelkan proses bisnis, membuat alur kerja.

Merepresentasi Algoritma

Sebelum desain dimasukkan ke dalam geometri sistem,  perancang sistem terlebih dahulu menentukan algoritma untuk menyelesaikan proyek.  Flowchart dibuat untuk memenuhi kebutuhan ini. Flowchart dimaksudkan untuk memberikan berbagai simbol yang dapat mewakili sistem yang  dibangun.

Memodelkan Proses Bisnis

Sebuah proses bisnis bukan hanya tentang keuntungan, itu berarti melibatkan berbagai tugas, dari yang sederhana hingga yang kompleks. Tujuan pemodelan diagram alur adalah untuk memberikan hasil yang dapat diprediksi dan konsisten.

Membuat Alur Kerja

Sebuah proses bisnis bukan hanya tentang keuntungan, itu berarti melibatkan berbagai tugas, dari yang sederhana hingga yang kompleks. Tujuan pemodelan diagram alur adalah untuk memberikan hasil yang dapat diprediksi dan konsisten. Flowchart berfungsi untuk menentukan integritas proses yang ada dan dapat menghasilkan hasil yang berkualitas tinggi berdasarkan prosedur.  Selanjutnya, kita memerlukan diagram alur untuk mendefinisikan dan mengelola alur kerja.

Mulai

Output hasil proses

Jika ya, selesai

Diagram flowchart ini menggambarkan prinsip pemikiran komputasional yaitu input, proses, dan output. Namun jika hasil belum benar, maka program akan mengulang proses dari awal.

28. Apa persamaan flowchart dan data flow diagram


Persamaan flowchart dan data flow diagram adalah memiliki fungsi yang sama, yaitu untuk mempermudah kita dalam menggambarkan alur suatu proses. Pada data flow diagram sendiri simbol bangun datar yang tersedia untuk menjelaskan suatu proses tidak sebanyak simbol bangun datar pada flowchart.

Pembahasan

Setelah mengetahui apa persamaan dari flowchart dan data flow, maka selanjutnya kita akan menyebutkan perbedaan dari flowchart dan data flow diagram, beberapa perbedaan antara flowchart dan data flow diagram adalah sebagai berikut

Alur proses yang dijelaskan menggunakan flowchart harus digambarkan secara urut, sedangkan pada data flow bisa digambarkan secara serentak.Penggunaan simbol pada flowchart lebih banyak daripada data flow diagram.Pada flowchart tidak terdapat tingkatan jenis flowchart, pada data flow diagram terdapat tingkatan.

Untuk membuat flowchart dan data flow, kita bisa menggunakan aplikasi seperti microsoft word, microsoft visio.

Program Penjumlahan, Pengurangan, dan Perkalian Menggunakan 4 Variabel

# program penjumlahan, pengurangan, dan perkalian menggunakan 4 variabel

a = 10

b = 5

c = 3

d = 2

# penjumlahan

hasil_jumlah = a + b + c + d

print("Hasil Penjumlahan:", hasil_jumlah)

# pengurangan

hasil_kurang = a - b - c - d

print("Hasil Pengurangan:", hasil_kurang)

# perkalian

hasil_kali = a * b * c * d

print("Hasil Perkalian:", hasil_kali)

Program Menghitung Luas dan Keliling Persegi Panjang

# program menghitung luas dan keliling persegi panjang

panjang = 6

lebar = 4

# menghitung luas

luas = panjang * lebar

print("Luas Persegi Panjang:", luas)

# menghitung keliling

keliling = 2 * (panjang + lebar)

print("Keliling Persegi Panjang:", keliling)

Program Rata-Rata Nilai

# program menghitung rata-rata nilai

nilai_mtk = 80

nilai_ipa = 90

nilai_tik = 85

# menghitung rata-rata

rata_rata = (nilai_mtk + nilai_ipa + nilai_tik) / 3

print("Rata-rata Nilai:", rata_rata)
