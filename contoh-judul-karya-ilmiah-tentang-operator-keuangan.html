13. Jelaskan contoh penggunaan operator logika!​


Jawaban:

Operator Logika merupakan bagian dari fungsi perbandingan yang digunakan untuk menunjang penulisan rumus, terutama pada rumus excel atau fungsi IF. Adapun operator logika if yang sering digunakan adalah sebagai berikut:

(=) : Sama dengan

(<) : Lebih Kecil

(>) : Lebih besar

(<= ) : lebih Kecil sama dengan

(>= ) : lebih besar sama dengan

(<>) : Tidak sama dengan

Operator logika di atas jika digunakan sesuai dengan penyataan logika yang benar, maka hasilnya TRUE (Benar) namun sebaliknya jika pernyataan logikanya salah maka hasilnya adalah FALSE (Salah). Jadi kita harus teliti dalam memahami pernyataan logika dan penentuan penggunaan operator logika yang cocok.

Biasanya operator logika di atas digunakan untuk penulisan rumus menggunakan fungsi IF karena pada fungsi IF terdapat pernyataan logika dan pilihan-pilihan sesuai kriteria yang dibuat.

17. apa pengertian dan contoh dari google syntax dan boolean operator


[jawaban]

Mata Pelajaran: Teknologi Informasi
Kata kunci: Google Syntax, Boolean Operator

• Jawaban pendek:

Apa pengertian dan contoh dari google syntax dan boolean operator

Google Syntax
adalah sintaks khusus yang digunakan pada search engine Google untuk "membatasi" area pencarian agar dapat menghindari adanya informasi yang tidak relevan dengan informasi yang kita butuhkan

Contoh Google Syntax:

1. ’intitle:’ 
untuk membatasi pencari hanya pada judul/nama web saja. Contoh: intitle:”matematika”.

2. ‘allintitle:’
untuk membatasi pencari hanya pada judul/web saja dengan variasi ekspresi pencarian lebih dari satu.
Contoh: allintitle:”matematika SMK” .

3. ’inurl:’
untuk membatasi pencarian hanya pada halaman web saja.
Contoh: inurl:p4tkmatematika.com/web/.


Boolean Operator
adalah fungsi yang digunakan untuk membandingkan dua nilai yang disebut dengan operator pembanding (comparison operator) atau operator relasi (relational operator). Hasil perbandingan dua nilai adalah nilai dengan tipe boolean (true dan false).

Contoh Boolean Operator:

1. Operator AND diletakkan di antara kata kunci yang gunakan dalam penelusuran, fungsinya adalah untuk mempersempit hasil pencarian. Misalnya, Filsafat AND Islam, akan mempersempit penelusuran, sehingga akan memperoleh hanya situs-situs yang mengandung kata Filsafat dan Islam di dalamnya.

2. Meletakkan operator OR di antara kata kunci berarti akan memperluas hasil pencarian . Misalnya, Filsafat OR Islam, akan menelusur situs-situs yang mengandung salah satu atau kedua kata tersebut di dalamnya.

3. Operator NOT juga untuk mempersempit hasil pencarian. Misalnya, Filsafat NOT Islam, berarti akan mencari situs-situs yang mengandung kata Filsafat, tetapi tidak mengandung kata Islam. (Beberapa mesin pencari menggunakan operator AND NOT)Tiap-tiap search engine memiliki cara yang berbeda dalam menerapkan teknik penelusuran dengan menggunakan operator Boole ini. Untuk mengetahuinya harus diperiksa pada petunjuk atau pedoman di masing-masing search engine.

2. Bagaimana Anda akan memformat komputer Anda?


Untuk memformat komputer kita perlu jendela XP CD. Boot CD dan ikuti petunjuk yang ditampilkan oleh komputer. Pilih drive yang diinginkan untuk menginstal sistem operasi. Berlari set baru file setelah selesai format drive.


3. Apa yang Anda akan lakukan dalam situasi jika komputer Anda dihentikan?


Ketika komputer dihentikan hal pertama adalah untuk menyimpan data. Satu-satunya pilihan adalah untuk me-restart komputer. Periksa bahwa semua komponen baik terhubung dan me-refresh memori 2 atau 3 kali. Selalu mencoba untuk menyimpan semua data sebelum menutup bawah komputer.

Beberapa istilah dasar komputer yang terdaftar di sini kernel, debugger, kompiler, control panel, memulihkan komponen, instalasi komponen dan seterusnya.


5. Bagaimana Anda akan menulis sebuah CD tanpa menggunakan Nero software?


CD drive dapat menulis tanpa menggunakan Nero software. Menulis CD hanya copy dan paste memerlukan data pada CD. Sekarang mengeluarkan CD drive dan lagi masukkan lagi. Komputer itu sendiri akan meminta Anda untuk menulis data pada CD sekarang masukkan perintah ya.

33. Jelaskan dan berikan contoh Operator Logical !


Jawaban:

Operator logika adalah sebuah operator yang digunakan untuk membandikan 2 buah nilai yang memiliki tipe data boolean. Tipe data boolean hanya memiliki 2 buah nilai yaitu true atau false.

Operator logika adalah AND, OR, dan NOT.

AND memiliki simbol " && "

OR memiliki simbol " || "

NOT memiliki sombol " ! "

Hasil mutlak dalam operasi logika : True ( T ) dan False ( F )

Operator AND  ( && ) :

T && T = T

T && F = F

F && T = F

F && F = F

Operator OR ( || ) :

T || T = T

T || F = T

F || T = T

F || F = F

Operator NOT ( ! ) :

!T = F

!F = T
