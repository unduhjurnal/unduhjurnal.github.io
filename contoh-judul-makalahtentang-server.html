1. server termasuk kedalam contoh komponen apa dan apa fungsi dari server​


Pengertian Server

Server atau dalam bahasa Indonesia biasa disebut peladen merupakan suatu sistem komputer yang memiliki layanan khusus berupa penyimpanan data. Data yang disimpan melalui server berupa informasi dan beragam jenis dokumen yang kompleks. Layanan tersebut ditujukan khusus untuk client yang berkebutuhan dalam menyediakan informasi untuk pengguna atau pengunjungnya.

Server berperan penting dalam menyediakan layanan akses lebih cepat untuk mengirim atau menerima data maupun informasi yang tersedia pada server. Dalam bentuk fisiknya, server berwujud jaringan komputer dan memiliki ukuran yang sangat besar dengan beberapa komponen pendukung prosesor dan RAM yang berkapasitas besar.

12. pengertian dan contoh hardware server dan brainware​


Jawaban:

Hardware adalah perangkat keras dari sebuah komputer yang dapat disentuh dan dipindahkan. Perangkat keras terdiri dari empat bagian yakni, perangkat bagian masukan (input), perangkat bagian keluaran (output), perangkat bagian pemrosesan (processing), dan perangkat bagian penyimpanan (storage). Contoh dari perangkat bagian input adalah mouse, keyboard, microphone, dan touch screen. Contoh dari perangkat bagian output adalah monitor, speaker, dan printer. Contoh dari perangkat bagian pemrosesan (processing) adalah prosesor yang berfungsi sebagai otak dari sebuah komputer. Contoh perangkat bagian penyimpanan (storage) adalah flashdisk dan harddisk.

Server adalah sebuah komputer yang menyediakan layanan untuk komputer lainnya dalam jaringan. Server dapat berupa perangkat keras atau perangkat lunak. Contoh dari server hardware adalah server blade, server tower, dan server rack.

Brainware adalah pengguna atau orang yang menggunakan sistem komputer. Tanpa adanya brainware, maka hardware dan software tidak akan bisa bekerja.

19. pengertian aplikasi client server serta contoh dan penjelasannya


Client-Server adalah pembagian kerja antara server dan client yang mengakses server dalam suatu jaringan.

Contohnya:

1. File Servers

File server vendors mengklaim bahwa mereka pertama menemukan istilah client-server.

Untuk sharing file melalui jaringan

2. Database Servers

Client mengirimkan SQL requests sebagai pesan pada database server,selanjutnya hasil perintah SQL dikembalikan.

Server menggunakan kekuatan proses yang diinginkan untuk menemukan data yang diminta dan kemudian semua record dikembalikan pada client.

3. Transaction Servers (Transaksi Server)

Client meminta remote procedures yang terletak pada server dengan sebuah SQL database engine.

Remote procedures ini mengeksekusi sebuah grup dari SQL statement

Hanya satu permintaan / jawaban yang dibutuhkan untuk melakukan transaksi

4. Groupsware Servers

Dikenal sebagai Computer-supported cooperative working

Manajemen semi-struktur informasi seperti teks, image, , bulletin boards dan aliaran kerja

Data diatur sebagai dokumen

5. Object Application Servers

Aplikasi client/server ditulis sebagai satu set objek komunikasi

Client objects berkomunikasi dengan server objects melalui Object Request Broker (ORB)

Client meminta sebuah method pada remote object

6. Web Application Servers (Aplikasi Web Servers)

World Wide Web adalah aplikasi client server yang pertama yang digunakan untuk web.

Client dan servers berkomunikasi menggunakan RPC seperti protokol yang disebut HTTP.

Server didukung dengan prosesor yang bersifat scalable dan RAM yang besar, juga dilengkapi dengan sistem operasi khusus, yang disebut sebagai sistem operasi jaringan (network operating system).Server juga menjalankan perangkat lunak administratif yang mengontrol akses terhadap jaringan dan sumber daya yang terdapat di dalamnya, seperti halnya berkas atau alat pencetak (printer), dan memberikan akses kepada workstation anggota jaringan. server dibagi menjadi beberapa jenis sesuai dengan fungsinya , berikut jenis server

Kemudian jelaskan cara kerja sebuah web server! 4. Jelaskan apa yang dimaksud dengan DNS Server! 5. Sebutkan dan jelaskan fungsi dari DNS Server!​


Jawaban:

1. Web server adalah sebuah program atau software yang berfungsi untuk menerima dan memproses permintaan dari browser web untuk menampilkan halaman web.

Dua contoh web server yang banyak digunakan saat ini adalah Apache dan Nginx. Apache adalah web server open-source yang sangat populer dan digunakan oleh sebagian besar situs web di seluruh dunia. Nginx adalah web server open-source yang dirancang untuk menangani lalu lintas web dengan efisien pada server modern yang sibuk.

2. Tiga contoh aplikasi web browser yang populer adalah:

Google Chrome: aplikasi web browser yang dikembangkan oleh Google. Fiturnya meliputi tab browsing, penjelajahan pribadi, dan integrasi dengan Google Drive dan aplikasi Google lainnya.Mozilla Firefox: aplikasi web browser open-source yang populer dengan fitur-fitur seperti tab browsing, penjelajahan pribadi, dan dukungan untuk add-on dan ekstensi.Microsoft Edge: aplikasi web browser yang dikembangkan oleh Microsoft. Fiturnya meliputi tab browsing, integrasi dengan Microsoft Office dan OneDrive, dan mode penghematan baterai untuk perangkat mobile.

3. Tiga contoh bahasa pemrograman yang digunakan untuk membuat web adalah:

HTML (Hypertext Markup Language): bahasa markah yang digunakan untuk membuat struktur dan konten halaman web.CSS (Cascading Style Sheets): bahasa stylesheet yang digunakan untuk mengatur tampilan dan format halaman web.JavaScript: bahasa scripting yang digunakan untuk membuat interaksi dan efek di halaman web.

Web server bekerja dengan menerima permintaan dari browser web melalui protokol HTTP (Hypertext Transfer Protocol) atau HTTPS (Hypertext Transfer Protocol Secure). Setelah menerima permintaan, server memproses permintaan tersebut dan mengirimkan kembali halaman web yang diminta ke browser.

4. DNS Server (Domain Name System Server) adalah sebuah server yang berfungsi untuk mengubah nama domain (seperti google.com) menjadi alamat IP yang sesuai (seperti 172.217.168.46) sehingga dapat diakses oleh komputer melalui jaringan internet.

5. Fungsi dari DNS Server adalah sebagai berikut:

Menerjemahkan nama domain menjadi alamat IP yang sesuai.Mengelola cache DNS untuk meningkatkan kecepatan akses ke situs web yang sering dikunjungi.Mencegah serangan DNS Spoofing atau DNS Cache Poisoning dengan memastikan bahwa alamat IP yang dikembalikan benar-benar berasal dari server DNS yang sah.

40. Apa itu server protokol, Dan contohnya


Penjelasan:

Aplikasi server merupakan sebuah aplikasi komputer dengan fungsi untuk melayani permintaan user terhadap permintaan akses yang berasal dari komputer client. Server sendiri merupakan sistem komputer dengan menyediakan berbagai jenis layanan yang telah ditentukan pada jaringan komputer.

Adapun, fungsi server itu sendiri dibagi menjadi 3 bagian penting, yaitu:

Menyediakan permintaan IP Address pada mesin komputer yang terhubung.

Menyediakan sebuah fitur demi keamanan perangkat komputer.

Menyimpan database dan aplikasi yang diperlukan pada komputer terhubung.
