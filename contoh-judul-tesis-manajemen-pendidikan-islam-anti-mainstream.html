Nama acara       : This is kimochi
Stasiun TV         : Ikeh TV
Waktu / Tanggal : 00.00 WIB, 30 Februari 2018.
Pembawa acara : Sora Aoi
Narasumber       : Kakek Legend
Tema                  : Sukses jadi aktor Bol-lybud
Pembahasan      : Tanpa melibatkan calon aktor/aktris film
Dialog                : 

P (Pembawa Acara)
N (Narasumber)

P : "Sejak kapan anda menjadi aktor Bol-lybud?"
N : "Sejak saya belum ada di dunia ini"
P : "Apa yang menyebabkan anda terjun di bidang perfilman?"
N : "Karena saya ingin jadi legend"
P : "Dimanakah produksi film terbaik menurut anda?"
N : "Menurut saya yang terbaik di Cuwiri Entertainment"
P : "Mengapa Anda ingin sekali menjadi aktor?"
N : "Selain menjadi legend, saya juga suka berolahraga walaupun sedikit gerakan yang saya bisa, dan hal itu memotivasi saya agar gerakan olahraga saya jadi banyak dan tubuh selalu sehat di waktu bekerja.
P : "Bagaimana proses ketika anda masih cecunguk hingga sekarang menjadi legend seperti ini?"
N : "Awalnya sih ya coba-coba ya, terus lama-lama kok enak.Ditambah ada rasa geli-geli ingin mutah gitu. Namun bayarannya banyak dan saya menikmati setiap hari pekerjaan saya"

Kesimpulan Dialog    : Tidak ada alasan untuk menjadi sukses. Cukup lakukan, nikmati, terus bertahan, dan jangan lupa untuk memakan kecambah. SALAM KIMOCHI

Jawaban:

Penjelasan:

Manajemen Pendidikan Islam adalah proses pemanfaatan semua sumber daya yang dimiliki (ummat Islam, lembaga pendidikan atau lainnya) baik perangkat keras maupun lunak.
