15. apa pengertian dan contoh dari possessive case


Possessive case adalah case dari suatu pronoun (kata ganti) ketika menunjukkan ownership (kepemilikan).

Contoh :
I had no time to shop for a camera, so I borrowed his camera.

18. sebutkan dan berikan contoh dari fungsi pada tombol change case


Change Case memiliki 5 bagian, yaitu:
1. Sentance Case = membuat huruf pertama menjadi hrf balok, dan huruf kedua dan seterusnya menjadi huruf kecil biasa
2. Lowercase = membuat semua huruf jadi huruf kecil
3. Uppercase = membuat semua huru jadi huruf balok
4. Capitalize Each Word = membuat huruf pertama setiap kata menjadi huruf balok
5. Toggle Case = membalikkan Huruf (huruf besar jadi huruf kecil, dan huruf kecil menjadi huruf besar)
itu semua akan berfungsi pada kata / kalimat yang di block sajaChange case = untuk mengubah penulisan huruf menjadi huruf besar, kecil, dsb

23. 1. a.Tuliskan kelompok change case dan jelaskan! b.berikan contoh masing-masing dari kelompok change case​


Penjelasan:

1.Change Case = berfungsi untuk membuat huruf depan jadi kapital, memperbesar huruf keseluruhan, memperkecil huruf.

2.sentence case, lowercase, uppercase, capitalize each word dan tonggle case. Fungsi dari masing-masing pilihan, yaitu :

1. Sentence case digunakan untuk membuat tulisan dengan huruf kapital di awal.

2. Lowercase digunakan untuk membuat tulisan dengan huruf kecil di awal.

3. Uppercase digunakan untuk membuat tulisan dengan huruf kapital semua.

4. Capitalize each word digunakan untuk membuat tulisan dengan huruf kapital di awal kata.

5. Tonggle case digunakan untuk membuat tulisan dengan huruf kecil di awal dan selanjutnya huruf kapital.
