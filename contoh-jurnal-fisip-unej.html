Dijawab :

A. Teknik yang digunakan ialah stratified random sampling, yakni suatu metode Pengambilan sampel secara acak dengan tetap memperhatikan persentase jumlah sampel yang diambil untuk tiap tiap kelompok terhadap jumlah populasi keseluruhan.

Alasan menggunakan metode Stratified Random Sampling ialah agar hasilnya tidak bias (melenceng) pada saat kesimpulan akhir, karena setiap sampel mendapatkan perlakukan yang sama namun tetap memperhatikan persentase tiap kelompok terhadap populasi keseluruhan

B. Jumlah sampel yang diambil untuk tiap tiap kelompok

Langkah 1

Mencari persentase masing masing kelompok terhadap populasi keseluruhan. Adapun rumus umum nya ialah:

%kelompok = (Jumlah anggota kelompok/populasi) x 100%

%sosiologi = (56/350) x 100%

                  = 0,16 x 100%

                  = 16 %

%Bahasa = (63/350) x 100%

               = 18 %

%administrasi = (70/350)x100 %

                       = (0,2) x 100 %

                       = 20 %

%hukum = (77/350) x 100 %

              = (0,22) x 100 %
