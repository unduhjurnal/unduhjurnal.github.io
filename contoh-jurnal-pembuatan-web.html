7. Apa yang dimaksud dengan web sites? Berikan beberapa contoh situs web yang kamu ketahui!


Ans:
Website (situs internet) adalah kumpulan dari beberapa web page (laman internet) yang terdiri dari konten dokumen teks maupun multimedia.

Website diidentifikasi dengan nama domain dan dipublikasikan dalam setidaknya satu server dan dapat diakses oleh entitas yang mengidentifikasi situs tersebut (entah dari IP publik atau dari jaringan LAN pribadi) dengan uniform resource locater/URL.

Contoh website dan jenisnya:
// indonesia.go.id (situs portal nasional yang diatur pemerintah Indonesia)
// google.com (situs layanan search engine populer)
// wikipedia.org (situs ensiklopedia digital populer)
// whitehouse.gov (situs resmi Gedung Putih yang diatur pemerintah Amerika Serikat)
// brainly.co.id (situs bertanya daring khusus untuk IP asal Indonesia)

Beda dan contoh dari web site dan web page ?

Perbedaan utama antara kedua adalah isi kontennya. Jika Web Page hanya menyediakan informasi tentang suatu topik, Website menawarkan banyak informasi tentang topik (Website ini difokuskan pada topik-nya yang biasanya terjadi).

• Penjelasan:

Web Page
Sebuah halaman web atau web page adalah sebuah dokumen atau informasi sumber daya yang cocok untuk World Wide Web dan dapat diakses melalui web browser dan ditampilkan pada monitor atau perangkat mobile.
Contoh: https://en.wikipedia.org/wiki/Earth

Website
Sebuah situs web (sering pula disingkat menjadi situs saja; web site, site) adalah sebutan bagi sekelompok halaman web (web page), yang umumnya merupakan bagian dari suatu nama domain (domain name) atau subdomain di World Wide Web (WWW) di Internet maupun. WWW terdiri dari seluruh situs web yang tersedia kepada publik. 
Contoh: https://www.wikipedia.org

27. siapa penbuat deep web dan dark web


Deep Web (juga dikenal dengan nama Deepnet, Darknet, Invisible Web, Undernet atau Hidden Web) di Indonesia dapat disebut sebagai Web Dalam. Deep Web merupakan bagian dari World Wide Web tetapi tidak termasuk ke dalam internet yang dapat dicari dengan mudah, yaitu dengan menggunakan mesin pencari web yang menggunakan indeks mesin pencari web. Deep Web berbeda dengan Dark Internet, dimana komputer tidak dapat lagi dijangkau melalui Internet, atau dengan Darknet yang merupakan jaringan untuk menukarkan data, yang dapat digolongkan sebagai bagian kecil dari Deep Web.

Mike Bergman, pendiri BrightPlanet yang memberikan istilah tersebut,mengatakan bahwa mencari di Internet pada saat ini dapat dibandingkan dengan menjaring ikan di permukaan laut: akan mendapatkan banyak ikan yang tertangkap di jaring, tetapi informasi yang sangat dalam tidak akan tertangkap. Kebanyakan dari informasi yang terdapat di jaringan Internet, terkubur sangat dalam situs yang dinamis, dan mesin pencari web yang standar tidak dapat menemukannya. mesin pencari web yang tradisional tidak dapat "mendeteksi" atau mengambil data di dalam Deep Web. Halaman-halaman tersebut dianggap tidak ada hingga mereka tercipta secara dinamis sebagai hasil dari pencarian secara spesifik. Sejak tahun 2001, Deep Web telah dinyatakan memiliki ukuran yang lebih besar dari Web biasa.
