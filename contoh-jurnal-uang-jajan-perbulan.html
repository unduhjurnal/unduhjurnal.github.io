28. Ratih mempunyai uang jajan Rp2.500 lebih banyak dari uang jajan Adi. Jika jumlah uang jajan mereka adalah Rp27.500, banyak uang jajan masing-masing adalah… (aljabar)​


27.500 - 2.500 = 25.000
25.000 : 2 = 12.500

Ratih : 15.000
Adi : 12.500

Jawaban:

asumsikan

uang Ratih = a

uang Adi = b

maka model matematikanya (bentuk aljabar) nya adalah

a = 2500 + b (pers. 1)

a + b = 27.500 (pers. 2)

SUBSTITUSI

pers. 1 kedalam pers. 2

2500 + b + b = 27.500

2500 + 2b = 27.500

2b = 27.500 - 2500

2b = 25.000

b = 25.000/2

b = 12.500

SUBSTITUSI

nilai b ke pers. 1 untuk mencari nilai a

a = 2500 + b

a = 2500 + 12.500

a = 15.000

KESIMPULAN

uang jajan Ratih = 15.000

uang jajan Adi = 12.500
