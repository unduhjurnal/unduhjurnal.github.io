4. Gaweya ukara kalimat kanti migunakake pola ukara J/W/L/K kang nganggo... a. katrangan panggonan b. katrangan wektu c.katrangan ubaya


♦ Pola J-W-L-Keterangan panggonan :

Nisa tuku buku ing toko buku

Nisa => JejerTuku => WasesaBuku => LesanIng toko buku => Katrangan panggonan

♦ Pola J-W-L-Keterangan wektu :

Nisa maca majalah isuk mau

Nisa => JejerMaca => WasesaMajalah => LesanIsuk mau => Katrangan wektu

♦ Pola J-W-L-Keterangan ubaya :

Nisa dolanan hape tapi kudu sinau

Nisa => JejerDolanan => WasesaHape => LesanTapi kudu sinau => Katrangan ubaya

==============================

Penjelasan :

★ Jejer = Subjek

★ Wasesa = Predikat

★ Lesan = Objek

★ Katrangan = Keterangan

Katrangan panggonan = Keterangan tempatKatrangan wektu = Keterangan waktuKatrangan ubaya = Keterangan syarat

===============================

Pembahasan :

Pola J-W-L-K iku kecakup saka »

Jejer. Manggon ing ngarep dewene ukara. Dadi subjek utawa sing nglakoni pakaryan.Wasesa. Manggon ing sak wise Jejer. Dadi predikat yaiku pakaryan e utawa kata kerja.Lesan. Manggon ing sak wise Wasesa. Dadi objek utawa sing dikenani pakaryan.Katrangan. Manggon ing mburi dewene ukara. Dadi keterangan tambahane ukara.

Tuladha Pola J-W-L-K (Panggonan) :

Adit mangan bakso ing warung

Adit = JejerMangan = Wasesa Bakso = LesanIng warung = Katrangan panggonan

Tuladha Pola J-W-L-K (Wektu) :

Dian sinau matematika ndek wingi

Dian = JejerSinau = WasesaMatematika = LesanNdek wingi = Katrangan waktu

Tuladha Pola J-W-L-K (Ubaya) :

Nabila mangan soto tapi kudu nyapu omah dhisik

Nabila = JejerMangan = WasesaSoto = LesanTapi kudu nyapu omah dhisik = Katrangan ubaya

Pendahuluan:

Peluang adalah perkiraan keberhasilan suatu kejadian secara teoritis. Peluang dirumuskan dengan:

[tex]P = \frac{n(A)}{n(S)} [/tex]

Dengan

n(A) adalah banyak kejadian yang diharapkan

n(S) adalah banyak semua kejadian yang mungkin terjadi

Untuk menghitung banyak kejadian gunakan kaidah pencacahan

Kaidah pencacahan adalah metode untuk menghitung banyak susunan yang memungkinkan. Metode dalam kaidah pencacahan:

Filling slots / pengisian tempat / kaidah perkalian. Prinsipnya mengalikan banyak kejadian yang mungkin dari tiap penyusunnyaPermutasi. Digunakan untuk menghitung banyak cara jika tidak ada pengulangan, dan dibulak-balik dianggap berbeda (misal "12" berbeda dengan "21"Kombinasi. Digunakan untuk menghitung banyak cara jika tidak ada pengulangan, dan dibulak-balik dianggap sama (misal terpilih siswa A dan B akan sama dengan terpilih siswa B dan A)
