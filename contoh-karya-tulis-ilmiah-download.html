4. Tuliskan langkah - langkah mendownload scracht​


Jawaban:

Langkah-langkah menginstal aplikasi Scratch

1. Download aplikasi scratch http://scratch.mit.edu/scratch2download/ Setelah selesai download aplikasi scratch klik 2 kali pada aplikasi yang telah terdownload.

2. Simpan data di data C:\Program Files (x86)\Scratch. Setelah itu klik next seperti gambar di bawah ini.

3. Setelah itu klik install seperti gambar di bawah ini.

4. Ketika aplikasi sudah terinstall, aplikasi scratch akan muncul seperti gambar dibawah pada desktop.

5. Selesai scratch sudah bisa digunakan untuk berkreasi.

Penjelasan:

(jadikan jawaban terbaik yaa)

Scratch adalah sebuah bahasa pemrograman visual untuk lingkungan pembelajaran yang memungkinkan pemula untuk belajar membuat program tanpa harus memikirkan salah-benar penulisan sintaks. Bahasa pemrograman ini dibuat oleh MIT Media Lab dari Massachusetts Institute of Technology.[1] Dengan Scratch, pengguna dapat membuat sendiri animasi, permainan, karya kesenian, dan lain-lain.[2]

Versi terbaru Scratch, Scratch 2.0 (atau Scratch 2), tersedia untuk diakses secara online atau dengan diunduh. Scratch 2.0 tersedia untuk sistem operasi Windows, Mac OS X, serta Linux.

Jawaban:

Penjelasan:

Contoh penerapan kurs beli dan kurs jual pada bank:

Sebagai contoh, jika seseorang ingin menukar rupiah dengan dolar Amerika Serikat (USD), bank akan menawarkan kurs beli dan kurs jual yang berbeda. Kurs beli adalah kurs yang digunakan bank saat membeli mata uang asing dari nasabah, sedangkan kurs jual adalah kurs yang digunakan bank saat menjual mata uang asing kepada nasabah. Contoh kurs beli bank adalah Rp13.500/USD dan kurs jual bank adalah Rp13.800/USD.

Contoh penerapan kurs beli dan kurs jual dalam kegiatan sehari-hari:

Contoh lain dari penerapan kurs beli dan kurs jual dapat dilihat pada transaksi valuta asing di money changer. Misalnya, jika seseorang ingin menukar rupiah dengan yen Jepang (JPY), money changer akan menawarkan kurs beli dan kurs jual yang berbeda. Kurs beli yang ditawarkan adalah Rp.9.000/JPY dan kurs jual yang ditawarkan adalah Rp.8.800/JPY. Pada contoh ini, seseorang akan mendapatkan nilai lebih besar jika menukar uang dengan kurs beli daripada kurs jual.

39. Sebutkan 5 contoh aplikasi bantu download dan jelaskan keuntungan mendownload dengan bantuan aplikasi tersebut​


IDM ( Internet Download Manager )

Dapat membuka Kecepatan Lebih dari 32 Koneksi Download ( tergantung bandwith nya )

ialah sebuah perangkat lunak (software) yang digunakan untuk mengunduh data atau file yang terdapat di internet dan juga meneruskan kembali (resume). Program ini diproduksi di New York, Amerika ini menduduki posisi teratas dalam memaksimalkan kecepatan pengunduhan data (download). Tampilan antarmuka (interface) yang sederhana menjadikan software IDM lebih bersahabat dengan user atau pemakainya. Seringkali kita memanfaatkan IDM untuk mendownload video, aplikasi, dan file lainnya seperti dari facebook, google drive, atau yang lainnya. Untuk mendownloadnya bisa mengunjungi link internet download manager, atau idm gratis versi idm 7.1.

Flash Get

Berbasis mirip seperti torrent hanya saja aplikasi ini bisa menggunakan link download, tidak ada konkesi download

software download manager yang cukup ternama dan kehadirannya pun terbilang sudah cukup lama, serta memiliki pengguna yang cukup banyak. FlashGet menggunakan Multi-server Hyper-threading Transportation (MHT), dimana fitur ini sudah mendukung beragam protokol dan mengatur dokumen dengan baik. Bahkan dengan fitur tersebut, proses download bisa meningkat lebih cepat hingga 10x lipat.

Selain itu, FlashGet juga sudah mendukung beberapa protokol ternama seperti HTTP, FTP, Bittorrent, eMule dan masih banyak lagi.

uTorrent

Aplikasi ini sangat cocok untuk file ukuran besar, dan torrent tidak bisa akses atau download dari link normal,

merupakan sebuah software download manager gratis yang dirancang khusus untuk Web terbaru (2.0). Orbit Downloader juga mampu mencegah proses download file yang mengandung spyware, adware, atau malware. Selain itu, Orbit Downloader memiliki kemampuan download file video/audio dari website streaming ternama seperti YouTube, MySpace, Imeem, dan Pandora. Selain itu, software download manager ini juga memudahkan para penggunanya untuk men-download file dari RapidShare.
