3. Untuk membuat desain PDF itu pada di aplikasi di PC/LAPTOP apa ?​


Untuk membuat desain PDF, Anda dapat menggunakan berbagai aplikasi di PC atau laptop, tergantung pada jenis desain yang ingin Anda buat. Berikut beberapa aplikasi yang umum digunakan:

1. Adobe Acrobat Pro: Ini adalah aplikasi khusus PDF yang sangat canggih, yang dapat digunakan untuk membuat, mengedit, dan mengelola file PDF. Ini ideal jika Anda perlu membuat PDF dengan fitur-fitur lanjutan.

2. Adobe InDesign: Jika Anda ingin membuat desain yang kompleks dan kemudian mengonversinya menjadi PDF, InDesign adalah pilihan yang baik. Biasanya digunakan untuk membuat buku, majalah, brosur, dan materi desain lainnya.

3. Microsoft Word: Word memungkinkan Anda untuk membuat dokumen yang dapat Anda konversi menjadi PDF. Ini baik untuk laporan, surat, atau dokumen teks sederhana.

4. Microsoft PowerPoint: PowerPoint adalah pilihan yang baik jika Anda ingin membuat presentasi visual dan kemudian mengonversinya menjadi PDF.

5. Adobe Illustrator: Jika Anda ingin membuat vektor grafis atau ilustrasi, Illustrator adalah pilihan yang baik. Anda dapat menyimpan karya Anda dalam format PDF.

6. Canva: Canva adalah alat desain online yang dapat digunakan untuk membuat berbagai jenis desain, termasuk poster, kartu ucapan, media sosial, dan banyak lagi. Anda dapat mengunduh desain Anda dalam format PDF.

7. Sketch: Ini adalah alat desain khusus Mac yang sering digunakan untuk desain antarmuka pengguna.

8. GIMP: GIMP adalah perangkat lunak grafis gratis yang mirip dengan Adobe Photoshop. Anda dapat membuat desain dan menyimpannya dalam format PDF.

9. CorelDRAW: Ini adalah aplikasi desain vektor yang digunakan untuk membuat ilustrasi, logo, dan desain lainnya yang dapat disimpan dalam format PDF.

Pilihlah aplikasi yang paling sesuai dengan kebutuhan Anda dan jenis desain yang ingin Anda buat. Semua aplikasi di atas dapat digunakan untuk membuat desain yang dapat diubah menjadi PDF.

16. Tuliskan Langkah-langkah mengubah format dokumen Ms. Word menjadi Pdf dengan aplikasi google dokumen!T_T ​


Jawaban:

Berikut adalah langkah-langkah untuk mengubah format dokumen Microsoft Word menjadi PDF dengan menggunakan aplikasi Google Dokumen:

1. Buka browser dan kunjungi situs Google Dokumen di https://docs.google.com.

2. Jika Anda belum login ke akun Google Anda, masuklah terlebih dahulu dengan menggunakan akun Google yang Anda miliki.

3. Setelah masuk ke Google Dokumen, klik tombol "Beranda" di kanan atas halaman.

4. Pilih "Upload" untuk mengunggah dokumen Word yang ingin Anda ubah menjadi PDF.

5. Pilih file dokumen Word yang ingin Anda ubah dan klik "Buka" untuk memulai proses pengunggahan.

6. Setelah dokumen berhasil diunggah, klik tombol "File" di bagian atas halaman.

7. Di menu drop-down, pilih "Unduh" lalu pilih "PDF Dokumen" untuk mengonversi dokumen menjadi format PDF.

8. Setelah memilih opsi PDF Dokumen, Google Dokumen akan memulai proses konversi dokumen Word menjadi PDF.

9. Setelah proses konversi selesai, dokumen PDF akan otomatis diunduh ke komputer Anda.

10. Buka file PDF yang diunduh untuk memastikan bahwa konversi selesai dengan benar.

Dengan mengikuti langkah-langkah di atas, Anda dapat mengubah format dokumen Microsoft Word menjadi PDF dengan menggunakan aplikasi Google Dokumen.

21. tuliskan langkah langkah membuat PDF di power point???by:leobastanta​


1. Pilih File > Ekspor.
2. Klik Buat dokumen PDF/XPS, lalu klik Buat PDF/XPS.
3. Dalam kotak dialog Terbitkan sebagai PDF atau XPS, pilih lokasi untuk menyimpan file. Jika ingin memberi nama lain, masukkan nama yang baru dalam kotak Nama file.
4. Secara opsional, jika Anda ingin mengubah tampilan akhir file PDF, lakukan salah satu hal berikut:

Di Optimalkan untuk, pilih Standar untuk kualitas lebih tinggi (misalnya, jika Anda ingin mencetaknya). Pilih Ukuran minimum untuk membuat file lebih kecil (misalnya, jika Anda ingin mengirimnya sebagai lampiran email).

Klik Opsi untuk mengatur tampilan presentasi sebagai PDF. Berikut adalah beberapa opsi yang tersedia:

Untuk menyimpan slide saat ini saja sebagai PDF, di bawah Rentang, pilih Slide saat ini.

Untuk menyimpan slide tertentu sebagai PDF, pada Slide, masukkan rentang nomor slide dalam kotak Dari dan Hingga.

Jika tidak ingin menyimpan slide tetapi ingin menyimpan komentar, handout, atau tampilan Kerangkanya saja, pada Terbitkan apa, pilih opsi yang sesuai. Anda juga dapat menentukan jumlah slide yang muncul dalam kotak menurun Slide per halaman, serta mengubah urutannya (Horizontal atau Vertikal).

Anda dapat memilih untuk menyertakan slide kerangka, slide tersembunyi, komentar, tinta, informasi non-cetak seperti properti Dokumen atau tag struktur Dokumen dalam PDF dengan memilih opsi yang sesuai.

Klik OK.

1. Perbedaan file PNG, PDF, CDR, dan file JPEG

PNG ( Portabel Network Graphics )adalah: -salah satu format penyimpanan citra yang

menggunakan metode pemadatan yang tidak

menghilangkan bagian dari citra tersebut.

Format PNG ini diperkenalkan untuk menggantikan format penyimpanan citra GIF. Secara umum, PNG dipakai untuk citra di Waring Wera Wanua.

PDF (Portable Document Format) adalah:

-sebuah format berkas yang dibuat oleh Adobe

Systems pada tahun 1993 untuk keperluan

pertukaran dokumen digital.

Format PDF digunakan untuk merepresentasikan dokumen dua dimensi yang meliputi teks, huruf, citra dan grafik vektor dua dimensi.

CDR ( CorelDRAW Graphics files ) adalah: -format untuk menyimpan informasi grafis, dibuat dalam program CorelDRAW.

Hal ini dapat berisi raster dan vektor gambar. Ini digunakan secara eksklusif dalam editor vektor dari perusahaan Corel. Untuk digunakan dalam perangkat lunak pihak ketiga itu perlu untuk menggunakan konverter. Versi 10 dan sebelumnya dapat dibuka di software vektor dari Adobe. CDR bisa dibuka juga dengan bantuan program Inkscape dan SK1.

Joint Photographic Experts Group (JPEG) merupakan:

-skema kompresi file bitmap.

Awalnya, file yang menyimpan hasil foto digital memiliki ukuran yang besar sehingga tidak praktis. Dengan format baru ini, hasil foto yang semula berukuran besar berhasil dikompresi (dimampatkan) sehingga ukurannya kecil.
