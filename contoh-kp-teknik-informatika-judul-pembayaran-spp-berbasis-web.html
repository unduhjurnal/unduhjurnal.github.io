11. biaya spp setiap semestar di suatu perguruan tinggi naik Rp50.000,00 dari spp sebelumnya. jika ada semestar ke-8 reno membayar spp sebesar Rp 1.750.000,00 total spp yang dibayarkan selama kuliah 8 semestar adalah​


Biaya SPP setiap semester di suatu perguruan tinggi naik Rp 50.000dari SPP bulan sebelumnya. Jika pada semestar ke-8 Reno membayar SPP sebesar Rp 1.750.000, total SPP yang dibayarkan selama kuliah 8 semester adalah Rp 12.600.000.

Penjelasan dengan langkah-langkah:

Pada soal di atas dikatakan biaya SPP yang dibayarkan setiap bulannya naik Rp 50.000 dari SPP bulan sebelumnya. Jadi biaya SPP yang dibayarkan setiap bulannya akan membentuk deret aritmatika dengan beda antar suku Rp 50.000. Pada soal diketahui SPP bulan ke-8 adalah Rp 1.750.000, jadi nilai dari U8 adalah Rp 1.750.000. Pertama kita cari terlebih dahulu nilai suku pertamanya. Suku ke-n barisan aritmatika akan memenuhi rumus:

Un = a + b ( n - 1 )

Jadi jika U8 = 1.750.000 dan b = 50.000, maka nilai a adalah:

U8 = a + b ( 8 - 1 )

1.750.000 = a + 50.000 ( 8 - 1 )

1.750.000 = a + 50.000 ( 7 )

1.750.000 = a + 350.000

a = 1.750.000 - 350.000

a = 1.400.000

Rumus untuk suku ke-8 dari barisan aritmatika tersebut adalah:

Un = 1.400.000 + 50.000 ( n - 1 )

Jumlah total SPP selama 8 bulan dihitung dengan rumus jumlah n suku pertama pada barisan aritmatika

Sn = [tex]\frac{n}{2}[/tex] ( a + Un )

S8 = [tex]\frac{8}{2}[/tex] ( 1.400.000 + U8 )

S8 = 4 ( 1.400.000 + 1.750.000 )

S8 = 4 ( 3.150.000 )

S8 = 12.600.000

Jadi total SPP yang dibayarkan selama 8 semester adalah Rp 12.600.000
