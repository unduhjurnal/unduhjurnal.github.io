Bagian-Bagian Cover

Cover skripsi, makalah atau laporan secara umum memiliki beberapa bagian atau komponen berikut ini:

22. tuliskan jenis pemrograman yang digunakan untuk mengenal struktur template​


Penjelasan:

HTML

Anda mungkin sudah sering mendengar HTML? Hyper Text Markup Language (HTML) merupakan bahasa komputer yang dimanfaatkan untuk membuat situs web. Dimana HTML terdiri atas berbagai kode pendek yang nantinya dijadikan sebuah teks untuk disimpan dalam file HTML tersebut. File inilah yang akan dibuka pada alat browser sehingga dapat membaca file untuk menerjemahkan kode HTML. Jenis editor HTM sederhana, misalnya Sublime, Atom, atau Notepad++.

PHP

Selain HTTP, ada juga PHP sebagai bahasa komputer paling populer dalam membuat situs web. Karena PHP berfungsi untuk mempercantik tampilan dari situs halaman web. Hanya dengan memanfaatkan PHP Anda bisa membuat fitur menarik untuk situs web, seperti login username dan password, membuat survey, mengecek form, dan lainnya. Anda dapat mengetahui sebuah web ada PHP, jika menemukan situs web yang diakhiri dengan PHP.

JavaScript

Ini dia jenis bahasa komputer yang sangat bermanfaat untuk membuat web baru. Bahkan berguna untuk melihat suatu animasi, tombol klik, dan pop-up pada website. Secara tidak langsung JavaScript memang berguna untuk menjadikan tampilan web semakin interaktif dan dinamis. Penggunaan JavaScript ini harus bersamaan dengan Node JS dan bahasa server-side sebagai kode pemrograman.

CSS

CSS atau kepanjangan dari Cascading Style Sheets yang memiliki tujuan untuk membuat halaman website. CSS disini lebih berfungsi untuk mengontrol aspek desain, seperti mengatur jenis font, warna tulisan, jarak spasi, dan lain sebagainya. paling tidak tampilan web akan tampak epik untuk berbagai perangkat berkat penggunaan CSS tersebut.

Jawa

Java sebagai bahasa pemrograman yang memiliki slogan terkenal, yakni menulis sekali, jalankan di mana-mana. Penggunaan Java ini biasa digunakan oleh big data developer, back-end developer, system engineer, Android developer dan architect. Menariknya lagi, Java Virtual Machine terbilang yang paling baik untuk dibuat secara kompatibel untuk platform mana pun.

Python

Python adalah bahasa komputer untuk memproses sebuah angka, teks, gambar, dan jenis data lainnya pada komputer. Python biasanya digunakan oleh website video YouTube, Google, NASA, dan New York Stock Exchange di setiap harinya. Berkat adanya Python, Anda dapat melakukan berbagai hal seperti membangun RSS Reader, memprogram web aplikasi CGI, membaca dan menulis MySQL, dan sebagainya.

Rubi

Ruby adalah bahasa pemrograman yang dirancang secara ramah oleh developer sehingga paling disukai oleh programmer. Ruby sendiri termasuk bahasa tingkat tinggi yang berguna untuk mendapatkan banyak hal hanya dengan baris kode bersih dan mudah dibaca. Kemudian kode Ruby tersebut tampak dalam bentuk kalimat yang ditulis dalam bahasa Inggris.

C++

C++ bisa dibilang sebagai bahasa komputer paling efektif karena sudah dibuat sejak tahun 1985. Performa dan reliabilitas dari C++ ini banyak dimanfaatkan oleh sistem seperti Microsoft, Adove, PayPal, dan Oracle. Hanya dengan C++ memungkinkan sebuah aplikasi desktop dapat dijalankan secara intensif. Namun dalam penggunaanya terbilang masih rumit dibandingkan dengan bahasa pemrograman lainnya.

Dasar visual

Visual Basic adalah bahasa pemrograman yang diluncurkan oleh Microsoft, dan menawarkan IDE Visual dalam pembuatan perangkat lunak berbasis sistem windows. Sebetulnya Visual Basic masih dalam satu keluarga dari bahasa komputer BASIC. Namun para developer lebih suka menggunakan Visual Basic dalam mengembangkan software-software tertentu.

33. Bagaimana cara memilih template template Microsoft publisher?


Pelajaran TI
=================
Setiap kali Anda membuka Publisher 2013 atau memilih File ➪ New [CTRL + N], Publisher akan menampilkan pilihan galeri New Template di Backstage View sehingga Anda bisa mencari dan memilih sebuah template untuk digunakan membuat file publikasi baru. Ketika komputer Anda terhubung dengan internet dan Sign In ke Publisher, Anda bisa klik Featured diatas thumbnail template dan Scroll/menggulung kebawah untuk melihat template lain yang tersedia untuk di-download. Klik thumbnail template untuk melihat jendela preview dan deskripsinya, lalu klik Create untuk mendownload template tersebut dan membuat publikasi baru menggunakan template tersebut.

JAWABAN :

Ketika menggunakan Microsoft publisher, untuk mengatur template warna adalah color schemes/ insert color.

PENJELASAN :

a.) Insert color akan membuka dialog pilih warna , yang memungkinkan kamu memilih nilai warna dan menyisipkannya ke dalam dokumen. Dialog memungkinkan kamu untuk melihat atau memilih warna dan kemudian memasukkan nilai warna ke dalam file aktif.

b.) color schemes digunakan untuk mengatur konfigurasi warna yang akan digunakan pada halaman slide, diantaranya konfigurasi warna untuk latar belakang (background), shadow, teks, title text, dan link.

Setelah berada di dashboard blog klik menu Template.

Cara Mengganti Template Blogspot_2

Masih di menu Template, scroll mouse Anda dan temukan sejumlah pilihan template yang sudah disediakan oleh Blogger.com. Ini yang saya maksudkan di paragraf awal, jika Anda ingin menggunakan template yang sudah ada silakahkan arahkan kursor ke salah satu template dan klik opsi Terapkan ke Blog.
