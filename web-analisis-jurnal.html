Jawaban & Penjelasan:

Hubungan antara tahapan Analisis dan desain untuk perancangan web terjadi secara iteratif melalui pembuatan prototipe. Di sini, persyaratan yang utama organisasi dan pengguna sistem didefinisikan dan diterjemahkan ke dalam desain dari mana sistem dapat dibangun.

tahapan analisis dan desain diikuti oleh fase implementasi, di mana penulisan program, membangun database, menguji sistem, dll. terjadi.

Diagram berikut menggambarkan hubungan antara ketiga fase tersebut bisa dilihat pada gambar yang saya berikan:

proses pemodelan adalah bagian penting dari analisis, yang digunakan untuk menganalisis arus informasi untuk mengoptimalkan proses bisnis. Analisis penyimpanan informasi dilakukan melalui pemodelan data.

tahapan desain pada dasarnya menentukan bagaimana sistem harus terstruktur atau bagaimana sebuah sistem informasi akan beroperasi. Target utama dalam tahapan ini termasuk mengembangkan desain yang berpusat pada pengguna, yaitu mengoptimalkan pengalaman pengguna sesuai dengan semua faktor.

  Implementasi mencakup aktivitas seperti pengujian dan peninjauan. Kegiatan ini terjadi setelah analisis dan desain selama pembuatan prototipe.

   Meskipun kegiatan analisis, desain, implementasi, dan pemeliharaan dianggap terpisah dalam siklus hidup pengembangan sistem, tetapi ada tingkat tumpang tindih yang besar di antara fase-fase ini dan semuanya terjadi secara bersamaan sebagai bagian dari pendekatan prototyping evolusioner.

   Sementara analisis dan persyaratan sedang berlangsung, desain dan implementasi akan terjadi secara bersamaan untuk menghasilkan storyboard dan prototipe.

   Setelah sistem hidup, pengukuran dan peninjauan akan dimulai. Segera setelah sistem hidup, maka akan diperlukan untuk melakukan pembaruan kecil secara terus menerus terhadap konten dan layanan.

   Untuk setiap pembaruan, proses pembuatan prototipe skala kecil yang melibatkan analisis, desain, dan pengujian akan dilakukan.

   Untuk pembaruan besar, analisis dan desain, pengembangan dan pengujian skala penuh lainnya akan diperlukan.


4. 1. Jelaskan apa tugas profesi dalam pengembangan aplikasi web berikut : a. Web Designer b. Web Programmer c. Web Administrator d. Web Master


Kelas: IX

Mata Pelajaran: Teknologi Informasi

Materi:  World Wide Web        

Kata Kunci:  web, http, internet

 

Tugas profesi dalam pengembangan aplikasi web berikut:


a. Web Designer: merancang susunan dan tampilan suatu situs pemrograman web


b. Web Programmer: membuat sistem dan halaman web sesuai dengan rancangan, dengan menggunakan bahasa pemrograman web


c. Web Administrator: menjaga kelangsungan kerja sistem web sehari-hari


d. Web Master: tugasnya adalah gabungan dari ketiga tugas di atas

 

Jawaban panjang:

 

a. Web Designer:

 

Ketika suatu web akan dibuat, maka disusun terlebih dahulu rancangan bentuk halaman, serta susunan alur halaman dalam bentuk link dari satu halaman ke halaman lain. Jika web menggunakan sistem halaman dinamik, maka database yang digunakan juga perlu dirancang

Merancang susunan dan tampilan suatu situs pemrograman web ini adalah tugas dari web designer.


b. Web Programmer:

 

Ketika sudah ada rancangan bentuk halaman dan alur situs, maka yang membuat sistem dan halaman web sesuai dengan rancangan, dengan menggunakan bahasa pemrograman web adalah Web Programmer. Bahasa pemrograman yang digunakan adalah HTML dengan dapat ditambah dengan Ruby, PHP atau ASP untuk logika halaman, CSS untuk menata tampilan halaman web, SQL untuk menghubungkan halaman web dengan database serta JavaScript untuk tampilan interaktif.


c. Web Administrator:

 

Ketika halaman web sudah jadi maka yang menjaga kelangsungan kerja sistem web sehari-hari adalah Web Administrator. Halaman web memerlukan orang yang rutin mengecek jalanya web apakah ada permasalahan atau tidak.


d. Web Master:

 

Web master tugasnya adalah gabungan dari ketiga tugas di atas. Web master harus memiliki skill yang dibutuhkan untuk melakukan desain, pembuatan dan perawatan situs web.

23. Web atau situs web memiliki halaman web yang berisi data dan informasi yang tersimpan di web server dalam bentuk​


﻿﻿Web atau situs web biasanya terdiri dari halaman-halaman web yang berisi data dan informasi dalam bentuk teks, gambar, video, dan elemen-elemen multimedia lainnya. Halaman-halaman web ini disimpan di server web dalam bentuk file HTML, CSS, JavaScript, dan file media lainnya.

﻿File HTML (Hypertext Markup Language) adalah dasar dari semua halaman web dan digunakan untuk membangun struktur halaman web. File CSS (Cascading Style Sheets) digunakan untuk memperindah tampilan halaman web dengan mengatur warna, font, dan layout. Sedangkan file JavaScript digunakan untuk membuat halaman web interaktif dan dinamis.

﻿Selain itu, halaman web juga dapat menyimpan data dan informasi dalam bentuk file media seperti gambar, video, dan audio. File-file media ini disimpan di server web dan dapat diakses oleh pengguna melalui halaman web.

﻿Secara umum, semua data dan informasi yang terdapat di halaman web disimpan di server web dalam bentuk file yang kemudian diakses oleh pengguna melalui browser web.

25. Apakah perbedaan antara web browser,web page,web server


Web browser, web page, dan web server adalah tiga istilah yang terkait dengan penjelajahan halaman di dunia maya yang dilakukan oleh pengguna internet.

WEB BROWSER ADALAH PERANGKAT LUNAK YANG DIPAKAI PENGGUNA INTERNET UNTUK BERSELANCAR DI DUNIA MAYA.

WEB PAGE ADALAH HALAMAN DI INTERNET YANG MENYAJIKAN INFORMASI KEPADA PENGGUNA DAN HANYA DAPAT DIBUKA DENGAN WEB BROWSER.

WEB SERVER ADALAH SISTEM ATAU PERANGKAT YANG DIGUNAKAN PIHAK PENYEDIA KONTEN UNTUK MENYAJIKAN ATAU MENAMPILKAN WEB PAGE DI INTERNET SERTA MENGAKOMODIR AKTIVITAS PENGGUNA DI WEB PAGE TERSEBUT.

37. Analisislah mana yang paling menguntungkan antara bank umum dan bank perkreditan rakyat .(Carilah contoh beberapa bank untuk dilakukan analisis kalian dapat membuka situs web dari bank yang akan kalian analisis dan Carilah informasi produknya lalu bandingkan dari kedua jenis bank tersebut) ​


Jawaban:

Antara bank umum dan bank perkreditan rakyat, lebih untung menggunakan bank umum. Alasan utamanya adalah, bank umum memberikan lebih banyak jasa perbankan ketimbang bank perkreditan rakyat. Adapun bank umum memberikan jasa dalam tabungan dana masyarakat, giro dan deposito, menyalurkan dana kepada masyarakat dalam bentuk kredit, transfer uang antar nasabah, jual beli valas, kartu kredit, dan lainnya. Sementara bank perkreditan rakyat hanya memberikan jasa pada tabungan dan deposito, menyalurkan dana kepada masyarakat dalam bentuk kredit, dan tidak ada jasa perbankan lainnya.

Penjelasan:

Di dunia perbankan yang ada di Indonesia, bank dibagi menjadi dua, yaitu bank umum dan bank perkreditan rakyat (BPR).

Bank adalah sebuah lembaga yang didirikan dalam bidang keuangan. Bank umumnya didirikan dengan memiliki beberapa kewenangan seperti menerima simpanan uang dari masyarakat, meminjamkan uang ke masyarakat, dan menerbitkan surat sanggup bayar.

Menurut PP NOMOR 70 TAHUN 1992 TENTANG BANK UMUM pada pasal 2 ayat 1, dijelaskan bahwa, jika seseorang ingin mendirikan bank di negara Indonesia, maka seseorang tersebut wajib menyetorkan atau memiliki modal sekurang-kurangnya Rp. 50.000.000.000 atau 50 milyar rupiah.

Bank umum memberikan jasa perbankan seperti tabungan dana masyarakat, giro dan deposito, menyalurkan dana kepada masyarakat dalam bentuk kredit, transfer uang antar nasabah, jual beli valas, kartu kredit, dan lainnya.

Bank Perkreditan Rakyat atau BPR adalah sebuah lembaga keuangan bank yang ada di Indonesia dan lembaga tersebut melaksanakan kegiatan usaha secara konvensional ataupun berdasarkan Prinsip Syariah. Bank Perkreditan Rakyat atau BPR dalam kegiatannya tidak memberikan jasa dalam lalu lintas pembayaran.

Bank Perkreditan Rakyat hanya memberikan jasa pada tabungan dan deposito, menyalurkan dana kepada masyarakat dalam bentuk kredit, dan tidak ada jasa perbankan lainnya. Jasa perbankan lainnya yang dimaksud adalah kegiatan dalam valuta asing, dan melayani jasa cek/giro (giralisasi). Bank Perkreditan Rakyat atau BPR hanya boleh beroperasi di dalam 1 (satu) propinsi.
